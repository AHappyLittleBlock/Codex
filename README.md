# Codex

This repository now includes two example Swift game projects for iPad.

* **`iPadFPSApp`** – a simple SceneKit first-person shooter demo where the player jumps from a helicopter, fights red enemies and is joined by NPC allies.
* **`iPadRoadTripApp`** – a SpriteKit-based road trip simulator with a day/night cycle, stats bars and obstacles. Animals appear at night and a summary of distance is shown on game over.

These projects are only starting points and do not represent full games.

See the respective `README.md` files in each folder for more information.
